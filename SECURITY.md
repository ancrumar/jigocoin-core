# Security Policy

## Supported Versions

Security updates are provided for the latest stable release of Jigocoin Core.

Users are strongly encouraged to run the most recent release available from the official repository.

## Reporting a Vulnerability

If you discover a security vulnerability affecting Jigocoin Core, please report it privately before public disclosure.

Do not open a public GitHub issue for security-sensitive reports.

## Disclosure Policy

Security reports will be reviewed and validated before disclosure.

When appropriate, fixes will be developed and tested privately before a public release is published.

## Response Time

The project aims to acknowledge security reports within a reasonable timeframe and to prioritize issues according to their potential impact on the network, node operators and users.

## Contact

Until a dedicated security contact address is established, security-related reports should be communicated directly to the project maintainers through the official GitHub repository.
