# Jigocoin Whitepaper v1.0

## 1. Introduction

Jigocoin (JGC) is a decentralized proof-of-work cryptocurrency inspired by the principles of **Seiryoku Zenyo** (maximum efficient use of energy) and **Jita Kyoei** (mutual prosperity and benefit), concepts developed by Jigoro Kano, founder of Judo.

The project seeks to combine the proven security and simplicity of decentralized blockchain networks with a philosophy that emphasizes efficiency, responsibility and long-term cooperation among participants.

Jigocoin is not designed merely as a speculative asset. It is intended as a practical and educational demonstration of how a decentralized monetary network can be built around values of sustainability, participation and shared progress.

---

## 2. Philosophical Foundations

### Seiryoku Zenyo

Seiryoku Zenyo can be translated as "the best use of available energy."

In the context of Jigocoin, this principle encourages:

* Efficient use of computational resources.
* Simplicity in network design.
* Avoidance of unnecessary complexity.
* Sustainable long-term operation.

Technology should serve a purpose. Complexity is only justified when it produces meaningful benefits.

### Jita Kyoei

Jita Kyoei can be translated as "mutual prosperity and benefit."

In the context of a decentralized network, this principle encourages:

* Open participation.
* Fair access to mining and validation.
* Cooperation between independent node operators.
* Long-term network resilience.

A decentralized system becomes stronger when participants benefit from the success of the network as a whole.

---

## 3. Network Architecture

Jigocoin operates as a peer-to-peer blockchain network.

Every full node maintains a complete copy of the blockchain and independently validates blocks and transactions according to consensus rules.

The network is composed of:

* Mining nodes.
* Full validation nodes.
* Public relay nodes.
* Wallet clients.

No central authority controls transaction validation, block production or monetary issuance.

The network remains operational as long as independent nodes continue to participate.

---

## 4. Consensus and Block Production

Jigocoin uses a Proof-of-Work consensus mechanism.

Miners compete to discover valid blocks by performing cryptographic calculations. The first miner to produce a valid solution broadcasts the block to the network.

Every node independently verifies:

* Block validity.
* Transaction validity.
* Consensus compliance.
* Monetary issuance rules.

Only blocks that satisfy all consensus requirements are accepted.

This process allows the network to maintain a shared and verifiable history without requiring trust in any central party.

---

## 5. Block Time and Network Efficiency

Jigocoin targets an average block interval of approximately one minute.

A shorter block interval provides:

* Faster transaction confirmation.
* More responsive network behavior.
* Improved usability for everyday transactions.

The selected block interval reflects the project's focus on practical efficiency while maintaining compatibility with proven blockchain security principles.

### Halving Schedule

The block subsidy is reduced by half every:

210,000 blocks

This predictable issuance schedule ensures that monetary creation decreases over time while preserving scarcity.

### Smallest Unit

1 JGC = 100,000,000 kan

The kan represents the smallest indivisible accounting unit of the network.
