# Jigocoin Whitepaper

## Overview

Jigocoin is an independent decentralized blockchain network derived from the Bitcoin protocol.

The project combines proven Bitcoin technologies with its own network identity, monetary policy and development philosophy.

## Consensus

Jigocoin uses Proof of Work consensus and maintains an independent blockchain.

## Monetary Policy

The network operates under its own issuance schedule and supply dynamics.

A significant portion of the historical supply became permanently inaccessible following incident JGC-0001, reducing the effective circulating supply.

## Network

Jigocoin maintains independent seed nodes, consensus nodes and mining infrastructure.

## Philosophy

The project is guided by:

* Seiryoku Zenyo
* Jita Kyoei

These principles influence technical development and long-term governance.

## Future Development

The objective is to build a sustainable, decentralized and independent monetary network while preserving interoperability with proven Bitcoin-derived technologies where appropriate.
