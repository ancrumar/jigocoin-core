# Jigocoin Philosophy

Jigocoin is guided by two principles derived from Judo:

## Seiryoku Zenyo

Maximum efficient use of energy.

Resources should be used efficiently, avoiding unnecessary waste while preserving security, decentralization and long-term sustainability.

## Jita Kyoei

Mutual welfare and benefit.

The network should create value for all participants: users, node operators, miners and developers.

## Long-Term Vision

Jigocoin aims to preserve the strengths of decentralized monetary networks while fostering cooperation, transparency and sustainable growth.

Although its origins trace back to the Bitcoin protocol, Jigocoin continues its evolution as an independent blockchain network.
