# Halving Schedule / Programa de Halving

**EN**

Block reward halves every:

210,000 blocks

Example:

Block 0–209,999 → 50 JGC
Block 210,000–419,999 → 25 JGC
Block 420,000–629,999 → 12.5 JGC

This continues until the maximum supply is reached.

**ES (España)**

La recompensa por bloque se reduce a la mitad cada:

210.000 bloques

Ejemplo:

Bloque 0–209.999 → 50 JGC
Bloque 210.000–419.999 → 25 JGC
Bloque 420.000–629.999 → 12,5 JGC

Este proceso continúa hasta alcanzar el suministro máximo.

# Monetary Philosophy / Filosofía Monetaria

**EN**

Jigocoin follows principles inspired by Jigoro Kano:

Seiryoku Zenyo — Maximum efficiency with minimum effort
Jita Kyoei — Mutual welfare and benefit

These principles guide the development of Jigocoin toward efficiency, cooperation, and sustainability.

**ES (España)**

Jigocoin sigue principios inspirados en Jigoro Kano:

Seiryoku Zenyo — Máxima eficiencia con el mínimo esfuerzo
Jita Kyoei — Prosperidad y beneficio mutuo

Estos principios guían el desarrollo de Jigocoin hacia la eficiencia, la cooperación y la sostenibilidad.
