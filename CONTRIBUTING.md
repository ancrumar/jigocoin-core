# Contributing to Jigocoin Core

Thank you for your interest in contributing to Jigocoin.

Jigocoin is an independent blockchain project focused on decentralization, efficiency, long-term sustainability, and the principles of Seiryoku Zenyo and Jita Kyoei.

## How to Contribute

Contributions are welcome in the following areas:

* Core protocol development
* Wallet improvements
* Mining infrastructure
* Documentation
* Testing and quality assurance
* Network monitoring and tooling

## Development Process

1. Fork the repository.
2. Create a dedicated feature branch.
3. Implement and test your changes.
4. Submit a pull request with a clear description of the modification.
5. Address review feedback if required.

## Coding Standards

* Follow the existing project coding style.
* Keep changes focused and well documented.
* Include tests when applicable.
* Avoid unrelated modifications in the same pull request.

## Testing

Before submitting a contribution:

* Build the project successfully.
* Run available unit tests.
* Run functional tests when relevant.
* Verify that the change does not affect network compatibility.

## Documentation

Documentation updates should accompany any significant functional change.

## Security

Do not publicly disclose security vulnerabilities.

Security-related issues should be reported privately to the project maintainers.

## License

By contributing to Jigocoin, you agree that your contributions will be distributed under the project's license.
