# Jigocoin Incident Registry

This document records significant incidents that have affected the Jigocoin network.

## Incident Classification

Incidents may include:

* Consensus failures
* Network outages
* Security vulnerabilities
* Infrastructure failures
* Critical software defects

Each incident record should include:

* Identifier
* Date
* Description
* Impact assessment
* Resolution
* Lessons learned
